📋 About The Project

This is an interactive dashboard built with Streamlit for visual analysis of DNA sequences. It allows researchers, students, and bioinformatics enthusiasts to explore genomic sequences intuitively, without needing to code.

\n Features:

✅ Sequence validation - Checks if the sequence contains only valid bases (A, T, C, G)

📊 Basic statistics - Length, GC%, base composition

📈 Interactive visualization - Composition charts and colored sequence preview

🔍 ORF search - Finds potential Open Reading Frames

📥 Multiple input methods - Manual typing, FASTA upload, or example sequence

💾 Download - Export the analyzed sequence



🛠️ Built With

- Python 3.9+ - Core programming language

- Streamlit - Dashboard framework

- Biopython - Biological sequence manipulation

- Pandas - Data analysis

- Matplotlib - Static visualizations

- Plotly - Interactive charts (if you added it)
